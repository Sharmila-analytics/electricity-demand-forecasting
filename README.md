# Electricity Demand Forecasting (ARIMA-LSTM)

## Overview
This project predicts electricity demand using a hybrid ARIMA-LSTM model.

## Tools & Technologies
- Python (Pandas, NumPy, TensorFlow)
- Time Series Forecasting
- Power BI

## Results
- Achieved 2.9% MAPE

## Key Features
- Hybrid model combining statistical and deep learning
- Forecasting future electricity demand
